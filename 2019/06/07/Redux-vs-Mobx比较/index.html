Redux vs Mobx比较

ybleeho
2019-06-07

1. 趋势


上图是npm近半年Redux和Mobx下载情况，使用Redux进行状态管理的还是大多数。

2. 风格

Redux Mobx
Functional Programming Object-oriented + Reactive programming
大量模版文件和代码 少量模版文件和代码
需要处理一步的中间件如thunk,saga 不需要thunk

3.状态管理

Redux Mobx
通常单一store 多个store
对象形式存储数据 可观察对象
状态对象不可变只读 状态对象可变重写
react-redux(Provider，connect) mobx-react(Provider, inject)
